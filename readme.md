# CloudArt - AI图像生成桌面客户端

基于Qt6的AI绘画工具，接入ComfyUI服务器，支持文生图、图生图、高清修复。

## 功能
- 会话持久化（SQLite）
- 实时进度（WebSocket）
- 参考图上传
- 历史画廊
- 精致深色UI + GIF动效
